# Golden Dawn Ritual Diagram And Implement Modes

## User Request

The user asked to continue inventing game modes for Golden Dawn rituals, diagrams, and painted items, explicitly mentioning the **Vault of the Adepti**.

This document expands the Golden Dawn branch beyond abstract correspondences into diagrammatic and painted ritual environments:

- Vault of the Adepti
- elemental tablets
- banners
- lamens
- grade diagrams
- altar layouts
- color-scale painting
- ritual furniture and implements
- Enochian-style diagram unlocks

## Design Principle

These should be **study-game modes**, not claims of ritual authority. The project should represent each item as:

- a visual object
- a data object
- a quiz surface
- a construction puzzle
- a source-labeled historical/esoteric artifact
- a possible game board for tarot/alchemy mechanics

Every mode should distinguish:

- Golden Dawn source layer
- later published/regardie-style layer
- OSOGD or modern teaching layer
- project house-rule/game layer

## Mode 1: Vault Of The Adepti Builder

### Player Fantasy

The player assembles the Vault as a colored ritual chamber and learns why every wall, color, symbol, and arrangement matters.

### Interface

- A rotatable polygonal room map.
- Clickable wall panels.
- Color palette with King/Queen/Prince/Princess scale toggles.
- Symbol stencil tray.
- Floor/ceiling toggle.
- "Reveal source note" side panel.

### Core Loop

1. Choose a wall or surface.
2. Select its required color scale or symbol family.
3. Paint or place the symbol.
4. Receive feedback.
5. Complete the chamber to unlock an Adept-level challenge.

### Quiz Types

- Place colors on the correct wall.
- Identify which wall is wrong.
- Restore missing planetary/zodiacal/emblematic labels.
- Match a wall to a source note.
- Compare two published diagrams and identify what changed.

### Game-Mode Variants

- **Vault Painter**: pure coloring/placement puzzle.
- **Vault Memory Walk**: player navigates wall by wall and names what they see.
- **Vault Repair**: the chamber has corrupted panels; fix them before the ritual timer ends.
- **Vault As Board**: alchemical/tarot blocks bounce or slide along walls with color-scale properties.
- **Adept Exam**: hidden labels; player must reconstruct the whole chamber from memory.

### Asset Needs

- wall panel
- floor tile
- ceiling tile
- color-scale swatch
- rose/cross emblem
- planetary/zodiacal glyph
- path/letter tag
- source-note card

## Mode 2: Elemental Tablet Constructor

### Player Fantasy

The player reconstructs a complex tablet from letters, elemental quadrants, colors, and names.

### Interface

- Grid board.
- Token bank of letters/symbols.
- Color overlays.
- Zoomed cell inspector.
- Error heatmap.

### Core Loop

1. Choose a tablet family.
2. Fill cells from a token bank.
3. Paint quadrants or subquadrants.
4. Validate rows, columns, names, and source labels.

### Quiz Types

- Put letters in the correct cell.
- Identify which quadrant a symbol belongs to.
- Trace a name through the grid.
- Hide every third cell and ask the player to restore it.
- Compare a simplified teaching diagram with a full tablet.

### Game-Mode Variants

- **Tablet Sudoku**: recover missing cells using row/column constraints.
- **Name Trace**: click letters in sequence to form a name.
- **Elemental Circuit**: route energy through correct elemental cells.
- **Tablet Minesweeper**: reveal cells without disturbing unstable names.
- **Tablet Print Shop**: generate blank worksheets and answer keys.

## Mode 3: Banner And Lamen Studio

### Player Fantasy

The player builds, paints, and tests ceremonial badges, banners, and lamens as symbolic machines.

### Interface

- Flat canvas.
- Shape stencil tray.
- Color scale selector.
- Inscription field.
- Source/reconstruction toggle.
- Export/print button.

### Core Loop

1. Select a ritual item.
2. Place required shapes.
3. Apply color scales.
4. Add letters, names, or symbols.
5. Test whether the item matches the selected tradition/source layer.

### Quiz Types

- Which color scale applies here?
- Which symbol is missing?
- Which inscription is a later addition?
- Which item belongs to which grade or ritual?
- Which version is intentionally wrong?

### Game-Mode Variants

- **Banner Painter**: recolor a ritual banner from memory.
- **Lamen Petal Drill**: place Hebrew letters onto the Rose Cross lamen.
- **Officer Badge Quiz**: match implements/badges to officer roles.
- **Inscription Puzzle**: restore fragmented divine names or mottos.
- **Print-Shop Mode**: output study cards, blank worksheets, and labeled diagrams.

## Mode 4: Grade Diagram Exam

### Player Fantasy

The player progresses through Golden Dawn-style grades by proving they can read and reconstruct diagrams.

### Interface

- Grade ladder.
- Diagram board.
- Token bank.
- "Hide labels" switch.
- Source note frame.
- Mastery meter.

### Core Loop

1. Choose a grade.
2. Receive a diagram challenge.
3. Place symbols, colors, names, or correspondences.
4. Explain or select why the placement is correct.
5. Unlock a harder diagram family.

### Quiz Families

- Neophyte: elements, quarters, basic Tree structure.
- Zelator: Earth/Malkuth, Hebrew-letter classes, alchemy basics.
- Theoricus: Air/Moon/Yesod, caduceus, tarot suits.
- Practicus: Water/Hod, magic squares, geomancy.
- Philosophus: Fire/Netzach, advanced element names, four worlds.
- Adept-style: Vault, Rose Cross lamen, color scales, pathwork.

## Mode 5: Color Scale Painter

### Player Fantasy

The player learns the Golden Dawn color scales by painting diagrams correctly.

### Interface

- Paint palette.
- Diagram masks.
- Swatch comparison.
- "source scale" selector.
- colorblind-safe labels.

### Core Loop

1. Select a diagram or object.
2. Choose a color scale.
3. Paint cells, paths, petals, or walls.
4. The game validates both hue family and symbolic category.

### Game-Mode Variants

- **Swatch Flashcards**: choose the correct color from several close options.
- **Path Painter**: paint Tree paths by attribution.
- **Vault Palette Test**: restore a corrupted wall color scheme.
- **Tarot Card Chromatics**: apply color-scale logic to attribution blocks.
- **Alchemy Reaction Color**: correct colors unlock block properties.

## Mode 6: Ritual Furniture Layout

### Player Fantasy

The player sets up the temple as a spatial puzzle.

### Interface

- Top-down room grid.
- Drag/click furniture icons.
- Direction compass.
- Officer/implement tokens.
- Ritual objective card.

### Core Loop

1. Read the ritual or grade setup prompt.
2. Place altar, pillars, banners, tablets, implements, and stations.
3. Validate orientation, adjacency, and symbolic logic.

### Game-Mode Variants

- **Temple Setup**: place objects before the ritual begins.
- **Error Spotter**: find misplaced furniture.
- **Officer Route Puzzle**: trace movement through the arrangement.
- **Balance Crossover**: ritual furniture becomes physical blocks on a tilting altar.
- **Level Builder Template**: export ritual room layouts.

## Mode 7: Diagram-To-Action Translator

### Player Fantasy

The player learns that diagrams are not inert pictures; they imply actions.

### Interface

- Diagram panel.
- Action-card bank.
- Timeline strip.
- Move readout.

### Core Loop

1. Click a diagram element.
2. Choose what action it implies: trace, vibrate, paint, place, invoke, banish, seal, circulate.
3. Build a ritual sequence from diagram evidence.

### Example Challenges

- Click a pentagram: choose tracing direction.
- Click a tablet quadrant: choose element and letter family.
- Click a Vault wall: choose color-scale or source note.
- Click lamen petal: choose Hebrew letter and attribution.

## Mode 9: Enochian And Spirit Diagram Unlocks

### Possible Modes

- Tablet grid reconstruction.
- Watchtower quadrant quiz.
- Name-tracing puzzle.
- Spirit-sigil print shop.
- Magic-square-to-sigil bridge.
- Error spotter for copied diagrams.
- Grade-gated unlocks from the Golden Dawn Explorer.

## Shared Mechanics Needed

### 1. Diagram Hit Regions

Every painted object needs semantic click regions:

- region id
- label
- expected symbol
- expected color
- source note
- game effect

### 2. Color Scale Data

Each color should be data, not hard-coded paint:

```js
{
  id: 'path-14-king-scale',
  object: 'tree-path',
  region: 'path-14',
  scale: 'king',
  color: '#...',
  source: 'placeholder'
}
```

### 3. Object Versioning

Each diagram/object may have multiple source versions:

- manuscript
- printed Regardie-style
- OSOGD teaching version
- project simplified version

### 4. Print-Shop Outputs

Ritual diagrams should export:

- blank worksheet
- answer key
- study card
- color swatch sheet
- object label card
- "spot the error" sheet
