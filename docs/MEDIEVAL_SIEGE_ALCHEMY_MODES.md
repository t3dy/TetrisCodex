# Medieval Siege Alchemy Modes

This document plans siege warfare variants and waits for approval before implementation.

## User Prompt

The user asked for a medieval siege warfare mode, perhaps tower defense, turn-based or real-time, and asked for plans for all versions and to wait before proceeding. The alchemist player character might throw bombs like Bomberman, use pool/lob-shot input, fire a flamethrower, cast an ice spell, and otherwise use alchemical effects.

## Core Premise

The player is an alchemist-engineer defending or attacking a medieval fortification. Alchemical substances and apparatus become siege tools:

- bombs
- mines
- Greek-fire-like streams
- freezing salts
- corrosive solvents
- smoke screens
- transmuting projectiles
- volatile vapors
- healing/preserving salves
- false apparitions

## Version A: Bomber Alchemist

Grid action mode inspired by bomb-placement games.

### Loop

1. Move through a castle/lab grid.
2. Place timed alchemical bombs.
3. Hide behind walls or use Salt shields.
4. Chain reactions through Sulphur, Mercury, solvents, and apparatus.
5. Open gates, destroy engines, or trap invaders.

### Alchemical Bomb Types

| Bomb | Effect |
| --- | --- |
| Sulphur bomb | Cross-shaped flame burst. |
| Mercury bomb | Vapor burst that phases through cracks. |
| Salt bomb | Crystallizes a temporary wall. |
| Solvent bomb | Dissolves weak stone or opens a moat channel. |
| Projection bomb | Casts force along a row. |
| Fermentation bomb | Delayed growth/rot hazard. |

## Version B: Tower Defense Athanor

Lane/route defense mode.

### Loop

1. Place alchemical furniture towers.
2. Enemies advance along roads, walls, tunnels, or moats.
3. Towers apply processes: calcine, dissolve, distill, separate, ferment, project.
4. Player routes matter and upgrades apparatus between waves.

### Tower Families

| Tower | Defense behavior |
| --- | --- |
| Crucible | Heat damage over time. |
| Solvent Jar | Slows armor and dissolves shields. |
| Alembic | Lifts/redirects vapors and flying enemies. |
| Separator | Splits enemy formations. |
| Fermenter | Turns corpses/debris into useful matter. |
| Projector | Long-range beam/cast. |

## Version C: Turn-Based Siege Laboratory

Tactics/puzzle mode.

### Loop

1. Each turn, choose a reagent, apparatus, or action.
2. Enemy engines advance or fire.
3. Reactions resolve in a predictable order.
4. Win by breaching a wall, preserving a gate, or transmuting a siege material.

### Strength

Best for historical framing because each level can teach one operation with a short source panel.

## Version D: Lob-Shot Artillery

Pool/Kirby-Dream-Course-like aiming applied to siege shots.

### Loop

1. Aim an arc or banked shot.
2. Choose projectile chemistry.
3. Fire into walls, towers, moats, or lab apparatus.
4. Collision and landing create cascade effects.

### Projectile Types

| Projectile | Behavior |
| --- | --- |
| Fire pot | ignites Sulphur/oil zones |
| Frost salt | freezes moat and fixes unstable matter |
| Mercury shell | bounces, splits, vaporizes |
| Acid jar | dissolves walls but contaminates water |
| Philosopher shot | rare transmutation objective |

## Version E: Flamethrower / Ice Spell Arcade

Real-time defense with directional tools.

### Loop

- aim streams
- manage fuel/heat/cooling
- combine fire, ice, solvent, and vapor
- prevent friendly lab equipment from exploding

## Historical-Thematic Anchors

This mode can discuss:

- practical alchemy and metallurgical knowledge
- gunpowder/firework traditions
- distillation and military/medical chemistry
- weapon salves and contested remedies
- fortress engineering, mining, and countermining
- the blurred boundary between craft, natural philosophy, and occult theory
