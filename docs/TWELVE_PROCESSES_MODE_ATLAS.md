# Twelve Alchemical Processes Mode Atlas

This document models the 12 zodiac-linked alchemical processes across the arcade's emoji-block mechanics.

## User Prompt

The user wants a separate website tab with ideas for each game mode showing how the 12 alchemical processes can be modeled within the limitations of a grid of emoji-labeled blocks.

## Core Constraint

The system is symbolic and mechanical. It cannot simulate full laboratory chemistry. It can, however, make process verbs playable:

- burn
- fix
- dissolve
- separate
- distill
- lift
- ferment
- multiply
- project

## The 12 Processes

| Zodiac | Process | Grid verb |
| --- | --- | --- |
| Aries | Calcination | Burn/reduce to ash. |
| Taurus | Congelation | Solidify/freeze/fix into shape. |
| Gemini | Fixation | Anchor, clamp, bind. |
| Cancer | Dissolution | Wet, loosen, dissolve. |
| Leo | Digestion | Warm over turns, ripen. |
| Virgo | Distillation | Extract, purify, move essence upward. |
| Libra | Sublimation | Lift, lighten, make volatile rise. |
| Scorpio | Separation | Split, cut, filter, divide. |
| Sagittarius | Ceration/Incineration | Soften with oil or burn intensely. |
| Capricorn | Fermentation | Rot, grow, transform over time. |
| Aquarius | Multiplication | Copy, spread, increase value. |
| Pisces | Projection | Cast influence outward, transmute target. |

## Mode Modeling Table

| Mode | Example process modeling |
| --- | --- |
| Balance Tetris | Calcination burns cells; Congelation increases friction; Projection casts a row; Multiplication increases slide-off score. |
| Rune Bloom | Processes become bloom modifiers: Fermentation grows, Separation breaks groups, Sublimation lifts blight, Projection spreads color. |
| Alchemy Snake | Eating an apparatus sets a process lens for later digestion. Distillation extracts a bonus; Dissolution grants wet/slip; Fixation slows and stabilizes. |
| Alchemical Three | Characters rotate/carry process furniture. Clamp locks blocks, Solvent Jar opens routes, Alembic changes what can pass. |
| VALIS Taxi | Processes become landing-pad operations: Distillation beam reveals cargo, Projection launches cargo, Dissolution makes pads slippery. |
| Wild Hunt | Processes classify or contain apparitions: Date+Distillation reveals, Fixation chronicles, Separation distinguishes rumor from record. |
| Alchemical Billiards | Apparatus are bumpers/pockets: Crucible heats, Separator splits balls, Projector scores distant pockets. |
| Alchemical Breakout | Bricks embody processes: hit Crucible to burn neighbors, Alembic redirects ball upward, Fermenter regrows bricks. |
| Alchemical Push | Process tiles modify pushed blocks: Solvent floors slide, Clamp stops, Mold freezes, Projector moves at range. |
| Lab Minesweeper | Processes are hidden hazards/clues: Calcination heat numbers, Dissolution corrosion clues, Fermentation delayed growth. |
| Siege Alchemy | Processes become weapons: Calcination bombs, Dissolution acid jars, Fixation barricades, Projection artillery. |

## Tutorial Ladder

Recommended order:

1. Salt/Sulphur/Mercury as basic matter.
2. Four easy processes: Calcination, Dissolution, Fixation, Projection.
3. Four medium processes: Distillation, Separation, Congelation, Multiplication.
4. Four advanced processes: Digestion, Sublimation, Fermentation, Ceration/Incineration.
5. Planet/metals layer.
6. Philosopher's Stone capstone.

## Example Capstone

Lead-to-gold level:

- Saturn/Lead block is heavy and difficult to move.
- Salt fixes it in place.
- Distillation and Separation remove impurities.
- Multiplication prepares tincture.
- Projection applies the Stone.
- The side frame explains the historical fantasy of transmutation and the gameplay simplification.
