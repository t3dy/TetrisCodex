# Alchemical Game World Bible

This is the organizing document for the broader alchemical game world. Rune Bloom and Balance Tetris are now treated as two games inside the same alchemical design universe, not as mutually exclusive directions.

## Core World Idea

The world is made of **emoji blocks**. Each block is a playable alchemical concept:

- an apparatus
- a substance
- a principle
- a planetary metal
- a zodiacal process
- an emblematic creature
- a stage of the Magnum Opus
- a volatile reaction product

The player learns the world by manipulating these blocks. Scholarship becomes rules.

## Primary Interaction Grammar

The main combinatorial grammar is:

```text
Zodiac Process + Optional Tria Prima Modifier = Spawned Alchemical Tile
```

Examples:

- Aries alone -> Crucible / Calcination furniture tile
- Aries + Sulphur -> combustible calcining substance
- Cancer alone -> Solvent Jar / Dissolution furniture tile
- Cancer + Mercury -> volatile dissolving solvent
- Taurus + Salt -> fixed coagulated matter
- Pisces + Mercury -> oily projected fluid

## Game Branches

### Balance Tetris

The platform game. Blocks have mass, position, and alchemical identity.

Key alchemical idea:

It can be profitable to tip substances off the correct side of the platform.

Current direction:

- planetary substances have proper exit sides
- proper side dumping scores
- wrong side dumping penalizes
- Mercury is profitable either way
- Salt anchors
- Sulphur burns matter to ash

### Rune Bloom

The garden puzzle. Blocks grow, bloom, react, and purify.

Key alchemical idea:

Alchemy becomes an ecology of placement, bloom, corruption, and transformation.

### Future Lab Game

The synthesis game. The player uses the zodiac/principle palette as the primary input method.

Key alchemical idea:

The player creates the game pieces through an alchemical keyboard.

## Block Ontology

### Apparatus Blocks

Zodiac alone creates apparatus/furniture.

These are stable board objects:

- Crucible
- Mold
- Clamp
- Solvent Jar
- Athanor
- Alembic
- Sublimator
- Separator
- Censer
- Fermenter
- Multiplier
- Projector

### Substance Blocks

Zodiac plus Salt/Sulphur/Mercury creates a substance:

- Salt: fixed, stable, anchored
- Sulphur: combustible, fiery, transformative
- Mercury: volatile, oily, fluid, wildcard

### Planetary Blocks

Planetary signs define side-affinity and material behavior.

Current Balance Tetris exit mapping:

| Planet | Sign | Proper Side | Game Meaning |
|---|---|---|---|
| Sun | ☉ | Right | solar, expansive, eastward |
| Moon | ☽ | Left | lunar, reflective, westward |
| Mercury | ☿ | Either | volatile, mediating |
| Venus | ♀ | Left | descending, receptive |
| Mars | ♂ | Right | fiery, active |
| Jupiter | ♃ | Right | expansive, benefic |
| Saturn | ♄ | Left | heavy, sinking |

### Emblem Blocks

Sourced from alchemical emblem literature:

- dragon
- toad
- salamander
- phoenix
- pelican
- golden apples
- hermaphrodite
- green lion

These should become rarer, more dramatic blocks.

## Source Discipline

Every major block should eventually have:

1. Game name
2. Emoji
3. Alchemical source term
4. Source text or scholarly note
5. Mechanical effect
6. Visual identity
7. Balance risk

## Design Rule

Names should never be merely decorative. If a tile is called an Alembic, it should separate, distill, extract, purify, or redirect. If a tile is called Salt, it should fix, preserve, anchor, or crystallize.
