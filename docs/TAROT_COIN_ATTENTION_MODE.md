# Tarot Coin Attention Mode

Tarot Coin Attention Mode turns a reading into a resource and map game. Every drawn card mints attribution coins. The player spends those coins as attention on connections between cards, earning points and lighting up a Tree of Life character sheet.

## User Prompt

The requested mode includes:

- whenever the user draws a card, they score a coin of each attribute
- earth, Sun, Aries, and other attributions are separate coin types
- coins can be spent as "attention" to connections between cards
- spending attention extends the reading and can score more points
- total elemental, planetary, and zodiacal progress lights up sections of the Tree of Life
- the Tree of Life should be shown as a small automap or zoomed-in character sheet
- the game needs several frames/screens:
  - drawing cards
  - textual readouts and interpretation questions
  - player steering choices such as career or relationships
  - a larger initiatory journey display
- long-term deck odds should respect a real tarot deck, where any card is one of 78

## Interpretation

This is a tarot reading game about attention as a scarce interpretive resource.

A card does not only mean something. It pays the player with symbolic currency:

- element coins
- planetary coins
- zodiac coins
- suit or Trump coins
- process coins

Those coins let the player choose what relationships deserve attention. A player who draws many Fire cards can spend Fire to argue for a line of heat, will, ordeal, or action. A player who draws Sun cards can spend Sun to clarify, crown, or illuminate a thread. A player who lacks the right coins may still force an interpretation, but at lower score or higher contradiction risk.

## Core Loop

1. Draw a tarot card.
2. Mint one coin for each displayed attribution.
3. Add those coins to the player's attention purse.
4. Choose a reading frame, such as career, relationships, study, health, or inner work.
5. Spend matching coins on card relationships.
6. Answer interpretation prompts generated by the chosen frame and the card attributions.
7. Score points for faithful, clever, or risky interpretations.
8. Add scored attribution progress to the Tree of Life sheet.
9. Light up Tree sections when enough linked attributions have been activated.
10. Continue the reading until the spread closes or the player runs out of useful attention.

## Attribution Coins

Every card can award several coins.

Example:

```js
{
  card: 'The Emperor',
  coins: ['Fire', 'Aries', 'Mars', 'Trump']
}
```

Coin classes:

| Class | Examples | Use |
| --- | --- | --- |
| Element | Fire, Water, Air, Earth | Pay for elemental sympathy, imbalance correction, or elemental readings. |
| Planet | Sun, Moon, Mercury, Venus, Mars, Jupiter, Saturn | Pay for dignity, temperament, timing, and Tree activation. |
| Zodiac | Aries through Pisces | Pay for pathwork, process mapping, and zodiac-linked alchemical operations. |
| Suit | Wands, Cups, Swords, Pentacles | Pay for practical suit-specific interpretation. |
| Card Class | Trump, Court, Pip | Pay for scale, agency, and role questions. |
| Alchemy Process | Calcination, Dissolution, Fixation, Projection, etc. | Pay for alchemical game-world integration. |

## Attention Spending

Attention is not generic mana. It is proof that the player has symbolic grounds for an interpretation.

### Elemental Sympathy

Cost:

- `1` coin matching either card's element

Effect:

- extend the reading by describing how the two cards share, clash, or temper an element
- score higher if the element is actually present

### Planetary Dignity

Cost:

- `1` planet coin from either card

Effect:

- extend the reading with a planet-flavored sentence
- light up related Tree regions
- score higher for coherent planetary framing

### Zodiacal Pathwork

Cost:

- `1` zodiac coin from either card

Effect:

- extend the reading through a zodiac or alchemical-process path
- stronger effect when the chosen frame naturally matches the sign

## Reading Frames

The player chooses a frame before spending attention.

| Frame | Reward | Risk |
| --- | --- | --- |
| Career | Rewards Sun, Mars, Saturn, Earth, Wands, Pentacles. | Water-heavy readings may need careful translation. |
| Relationships | Rewards Venus, Moon, Water, Cups, Libra, Taurus. | Harsh Mars/Saturn cards can create contradiction risk. |
| Study | Rewards Mercury, Air, Swords, Virgo, Gemini. | Fire-heavy answers may over-act before understanding. |
| Inner Work | Rewards Moon, Saturn, Trump, Water, Pisces, Scorpio. | The reading may become slower but deeper. |
| Alchemical Work | Rewards process coins and Tria Prima language. | Requires more exact source and process labeling. |

| Chokmah | Zodiac, force, Fire | dynamic pressure |
| Binah | Saturn, form, limit | structure and discipline |
| Chesed | Jupiter, mercy, expansion | generous interpretation |
| Geburah | Mars, severity, rupture | ordeal and correction |
| Tiphareth | Sun, harmony, gold | central synthesis |

Lighting rule:

- each scored connection adds progress to nodes linked to its coins
- node glow level increases at `1`, `3`, and `5` activation points
- the character sheet shows total initiatory journey, not just the current spread

## Screen Frames

### Draw Frame

Purpose:

- draw a card
- show card odds
- show minted coins
- show current spread line

### Reading Frame

Purpose:

- ask interpretive questions
- offer frame choices
- show consequences of attention spending
- preserve text readouts

### Attention Purse

Purpose:

- show current coin counts
- show which connection actions are affordable
- teach attribution literacy through resource constraints

### Tree Automap

Purpose:

- small always-visible Tree showing current glow
- lets the player see which initiatory zones are awakening

### Tree Detail

Purpose:

- zoomed-in character sheet
- shows node names, attributions, progress, and current interpretation bonuses

### Journey Log

Purpose:

- permanent textual history of draws, coins, choices, contradictions, and breakthroughs

## Scoring

Base scoring:

- draw card: `+1`
- spend matching attention: `+4`
- spend partly relevant attention: `+2`
- force unsupported connection: `-2`
- complete a three-card line: `+5`
- light a Tree node: `+3`

Frame bonus examples:

- Career + Sun/Mars/Saturn/Earth: `+2`
- Relationships + Venus/Moon/Water/Cups: `+2`
- Study + Mercury/Air/Swords: `+2`
- Inner Work + Moon/Saturn/Trump/Water: `+2`
- Alchemical Work + process coin: `+2`

## Design Notes

This mode should make the player feel that interpretation is earned. The player is not merely choosing from vibes. They are spending attention backed by visible symbolic evidence.

The design should stay flexible. Some tarot systems use different attributions. The game must eventually label the tradition used for every deck and spread.
