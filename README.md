# Rune Bloom

A separate original browser game built in this workspace.

Rune Bloom is a crystalline tile-placement puzzle. Grow outward from the center seed, place glyph-shaped pieces, bloom same-color horizontal or vertical runs of four, and keep spreading blight from sealing the garden.

## Controls

- Mouse: move the glyph preview, click to place
- Mouse wheel or right-click: rotate
- Keyboard: arrow keys move, `Q` / `E` rotate, `Space` / `Enter` place, `R` reset

## Goal

Reach 220 points before no legal placements remain. Every three turns, blight spreads. Blooms clear rune cells and purify adjacent blight.

## Run

Open `index.html` directly in a browser. No build step or dependencies are required.
