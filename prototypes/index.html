Prototype Lab

Self-contained mechanic experiments. Each is a single HTML file with stripped-down rendering and one experimental rule. Pick a feel that works before we port it into the main game.
PROTOTYPE 1
Spells
inspired by Tricky Towers
Earn spell cards every 5 pieces. Apply to next piece for game-changing effects.
← → move • ↑ rotate • space drop • 1/2/3 cast
PROTOTYPE 2
Bombs
inspired by Boom Blox
Bomb pieces tick down 5 turns then explode, blasting a 3×3 area of cells off the platform.
← → move • ↑ rotate • space drop
PROTOTYPE 3
Balloons
inspired by World of Goo
Balloon pieces have negative weight — placed high up, they pull the platform UP, counteracting heavy bottoms.
← → move • ↑ rotate • space drop
PROTOTYPE 4
JenGone
inspired by Jenga / Boom Blox demolition
Inverted gameplay. Tower starts pre-built. You drop anti-pieces that DELETE cells. Don't tip it.
← → move • ↑ rotate • space drop
PROTOTYPE 5
Zen
inspired by mobile stackers
Pure minimalism. Monochrome, no UI, no scores. Just blocks, gravity, and the platform's quiet hum.
← → move • ↑ rotate • space drop
